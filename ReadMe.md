# 🐾 POKA Friends – 포카(POKA) 브랜드 스토리 웹사이트

> 캐릭터 브랜드 **‘포카(POKA)’** 의 탄생 비하인드와 세계관을 감성적으로 전달하는 스토리텔링 기반 브랜드 웹 프로젝트

---

## 📌 About Project

**POKA Friends**는 단순한 캐릭터 소개가 아닌,  
포카(POKA)의 탄생 과정과 브랜드 감성을 웹으로 시각화한 **스토리 중심 브랜드 프로젝트**입니다.

브랜드 세계관(Storytelling), 감성적 UI/UX, 캐릭터 비주얼을 결합하여 사용자와 정서적 연결을 만드는 경험을 목표로 제작했습니다.

또한 본 프로젝트는 **기획 → 디자인 방향성 설정 → AI 협업 퍼블리싱 → GitHub Pages 배포**까지 1인 제작 방식으로 진행되었습니다.

---

## ✨ Key Features
